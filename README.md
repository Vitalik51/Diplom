# Pet Worlds RPG

Pet Worlds RPG — це навчальна гра на Python у стилі текстово-візуальної RPG з виглядом від першого лиця.

## Суть гри

Гравець має питомця та подорожує між світами.  
Для проходження гри потрібно знаходити картки-команди, отримувати досвід, прокачувати питомця та проходити тести для відкриття нових світів.

## Можливості

- вигляд від першого лиця;
- система питомця;
- рівні та досвід;
- картки-команди;
- навчальні тести;
- перехід між світами;
- просте керування.

## Керування

- W — рух вперед
- A — поворот ліворуч
- S — рух назад
- D — поворот праворуч
- E — підібрати картку
- Q — пройти тест
- C — список команд
- ESC — вихід
